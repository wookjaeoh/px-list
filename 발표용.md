# 매점 사전주문 웹사이트 제안 발표 구성

1. **제안 개요**
   - 목적: 쉬는시간 매점 이용으로 인한 수업 지각과 이동 혼잡 완화
   - 방식: 학생 사전주문, 매점 사전준비, 교사 운영 모니터링 구조
   - 성격: 학교 생활지도와 매점 운영 효율을 함께 개선하는 운영 시스템

2. **현재 운영상 문제**
   - 쉬는시간에 학생들이 1층 매점으로 집중 이동
   - 상품 선택, 대기, 결제 과정에서 쉬는시간 내 복귀가 어려움
   - 다음 수업 시작 지연, 복도 혼잡, 안전관리 부담 발생
   - 담임교사가 학생별 매점 이용 현황을 파악하기 어려움

3. **문제의 원인 분석**
   - 매점 이용 절차가 모두 현장에서 동시에 발생
   - 학생 이동 시간과 주문 대기 시간이 분리되지 않음
   - 매점주인은 주문 수요를 사전에 예측하기 어려움
   - 교사는 주문 가능 시간과 이용 현황을 통제할 수단이 부족함

4. **개선 방향**
   - 상품 선택과 주문은 교실 또는 개인 기기에서 사전 처리
   - 매점은 주문 목록을 미리 확인하여 상품을 준비
   - 결제와 수령은 매점에서 짧게 처리
   - 교사는 주문 가능 시간과 반별 이용 현황을 관리

5. **사용자별 역할**
   - 학생: 상품 확인, 주문 제출, 수령 시 현장 결제
   - 매점주인: 주문 목록 확인, 상품 준비, 수령 상태 처리
   - 담임교사: 본인 반 주문 현황 확인, 생활지도 자료 활용
   - 관리교사: 전체 운영 설정 및 계정 관리

6. **학생 이용 절차**
   - 학번과 비밀번호로 로그인
   - 판매 중인 상품만 확인
   - 주문 전 최종 확인 후 제출
   - 주문 후 변경 및 취소 불가
   - 매점에서 학번, 이름 또는 주문번호 확인 후 현장 결제 및 수령

7. **매점주인 관리 절차**
   - 학생 화면과 분리된 별도 주소로 접속
   - 주문 시간순 목록 확인
   - 주문자 학번, 이름, 상품명, 수량, 금액 확인
   - 주문 상태를 접수, 준비완료, 수령완료로 관리
   - 상품명, 가격, 판매여부 관리

8. **교사 관리 절차**
   - 학생 화면과 분리된 별도 주소로 접속
   - 주문 가능 시간을 열고 닫음
   - 본인 반 주문 현황 모니터링
   - 학생별 주문 횟수, 주문 금액, 선호 상품 등 통계 확인
   - 필요 시 생활지도와 학급 운영 자료로 활용

9. **권한 분리와 접근 통제**
   - 학생 화면에는 교사와 매점주인 기능 미노출
   - 학생 주소, 매점주인 주소, 교사 주소를 분리
   - 학생은 본인 주문만 조회 가능
   - 매점주인은 주문 처리에 필요한 정보만 확인
   - 교사는 본인 반 중심으로 현황 확인

10. **개인정보 관리 원칙**
    - 학생 계정은 교사가 일괄 생성
    - 로그인은 학번과 비밀번호 기반으로 운영
    - 비밀번호는 평문 저장 금지
    - 주문 목록에는 업무상 필요한 학번과 이름 중심으로 표시
    - 권한별 접근 범위를 명확히 제한

11. **운영 정책**
    - 결제는 온라인 결제가 아닌 매점 현장 결제로 시작
    - 주문 후 취소 불가 원칙 적용
    - 교사가 주문 가능 시간을 통제
    - 미수령 주문은 별도 확인 및 통계화
    - 초기에는 재고 관리보다 상품명, 가격, 판매여부 관리에 집중

12. **기대 효과**
    - 학생의 매점 체류 시간 감소
    - 수업 시작 지연 감소
    - 쉬는시간 복도 및 매점 혼잡 완화
    - 매점의 사전 준비 가능으로 운영 효율 향상
    - 교사의 학급별 이용 현황 파악 가능

13. **시범 운영 계획**
    - 1단계: 한 학급 또는 한 학년 대상 시범 운영
    - 2단계: 쉬는시간별 주문량, 수령 시간, 미수령 사례 측정
    - 3단계: 학생, 교사, 매점주인 의견 수렴
    - 4단계: 운영 정책 보완 후 확대 적용 검토

14. **검토해야 할 운영 기준**
    - 주문 가능 시간 설정 기준
    - 미수령 주문 처리 방식
    - 학생 계정 생성 및 비밀번호 재설정 절차
    - 담임교사와 관리교사의 권한 범위
    - 시범 운영 후 확대 여부 판단 기준

15. **도입 제안**
    - 즉시 전면 도입보다 제한된 범위의 시범 운영을 제안
    - 기술 도입보다 생활지도와 학교 운영 개선 관점에서 접근
    - 관리자, 교사, 매점주인의 역할을 분리하여 책임 범위 명확화
    - 운영 데이터 확인 후 학교 전체 적용 여부 결정
